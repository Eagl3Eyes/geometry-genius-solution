Question & Answers

1. What is DOM? What is the purpose of DOM?

Ans: DOM: The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web. The purpose of DOM: The purpose of the DOM is to provide developers with a consistent, standardized way to interact with Web documents, enabling them to create rich, dynamic Web pages and applications that adapt to user needs and preferences.

2. How can i select element from DOM?

Ans: getElementsByClass(), getElementById(), querySelector()

3. What is event bubble?

Ans: Event bubbling is an event that occurs when an event is triggered on a specific element of the Document Object Model (DOM) and the event is then propagated or "bubbled up" to parent elements in the DOM hierarchy, all the way up to the top-level Document object.

What is call back function? How does it works?

Ans: A callback function is a function that is passed as an argument to another function.

The basic idea of a callback function is that it allows you to define custom behavior that can be executed when a certain event occurs or some other task is completed. For example, in JavaScript, you can use a callback function to run some code after a web page has finished loading, or to perform some other asynchronous operation.
